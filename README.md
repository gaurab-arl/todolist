# Daily Planner

A responsive task management and scheduling web app. Add tasks with priority levels, filter them, and schedule your day with a built-in timetable — all in the browser, no backend needed.

---

## Features

- Add tasks with High / Medium / Low priority
- Filter tasks by priority
- Timetable section — schedule tasks with start and end times
- Dark mode toggle
- Motivational message display
- Modular JavaScript architecture (`js/app.js` as entry point)
- Fully responsive layout
